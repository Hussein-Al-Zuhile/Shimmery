ShimmerConfiguration

data class ShimmerConfiguration(val shimmerType: ShimmerType = ShimmerType.WITH_ALPHA_AND_GRADIANT, val gradientType: GradientType = GradientType.HORIZONTAL, val shape: Shape = RoundedCornerShape(16.dp), val gradientAnimationSpec: DurationBasedAnimationSpec<Float> = tween(1200), val alphaAnimationSpec: DurationBasedAnimationSpec<Float> = tween(1000))

This class is used to provide the configuration for the shimmer effect. You can use the shimmerConfiguration function to create a ShimmerConfiguration object.

Parameters

shimmerType

is used to determine the type of shimmer effect to be used.

gradientType

is used to determine the type of gradient to be used.

shape

is used to determine the shape of the shimmer effect.

gradientAnimationSpec

is used to determine the animation spec for the gradient.

alphaAnimationSpec

is used to determine the animation spec for the alpha.
